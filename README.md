# Movie Selection Project

A web-based application that allows users to browse, select, and explore movies based on different categories and preferences. This project focuses on improving user interaction and frontend-backend integration.

## Tech Stack
- Python
- Django
- HTML
- CSS
- JavaScript
- Database: SQLite
- Tools: Git, GitHub

## Features
- View and explore movie listings
- User-friendly interface for movie selection
- Dynamic content rendering using Django

## How to Run
1. Clone the repository
2. Install required dependencies
3. Run the Django server
